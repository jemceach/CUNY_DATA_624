Homework Process

This is the folder for the first of our two group homework assignments

I have created a folder for each week with homework assigned. You should drop your homework into the appropriate weeks folder

File Formats

Either .r or .Rmd is fine, so long as you include any data not included in the packages with it so we can all reproduce the problems in building the final submission

Naming Conventions

if you name your homework for each week by your 'first name-week', then we can track who's doing what and ask the right people for help/clarification or to use pieces and parts

Example:

bethany-one.Rmd

bethany-one.R

Submission File

As we agreed in our first group meeting, the submission for each of the homeworks (and possibly group projects depending ont he scope) will be an r bookdown document in PDF format which Professor Burke can access from anywhere!
